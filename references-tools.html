1. Data Modelling Tools and Studios

License

Hackolade (by IntegrIT SA/NV)

Hackolade was specially adapted to support the data modeling of Cassandra, including UDT and the concepts of Partitioning and Clustering keys. It lets users define, document, and display Chebotko physical diagrams.

https://www.hackolade.com/nosqldb.html#cassandra
Free Trial

KDM (by Datafluent)

KDM is a big data modeling tool specialized for Apache Cassandra which allows you to go through the query driven methodology to go from a conceptual data model to an optimized physical data model.

https://www.datafluent.org
Free Trial

AppStax (by Datastax)

AppStax is a fully managed and highly intelligent data service that gives you all the power and agility of a modern database without any of the headaches.
 

https://get.datastax.com/appstax/
Freeware

Dev center (by DataStax)

DataStax DevCenter is a free visual schema and query environment to create and run Cassandra Query Language (CQL) statements against Apache Cassandra and DataStax Enterprise.

https://downloads.datastax.com/#devcenter
OSS

JDL Studio (by JHipster)

DL-Studio, a tool for drawing JHipster JDL diagrams based on the JDL syntax. Import those file in JHipster to generate working apps. Try and edit the code on the left and watch the diagram change.

https://start.jhipster.tech/jdl-studio/
License

DBeaver EE (DBeaver-CORP)

DBeaver is free universal SQL client/database tool for developers and database administrators. It can work with any database server which has JDBC or ODBC driver. It supports pluggable extensions..

https://marketplace.eclipse.org/content/dbeaver-...
License

DataGrip (by Jetbrains)

DataGrip is the multi-engine database environment. If the DBMS has a JDBC driver you can connect to it via DataGrip. It provides database introspection and various instruments for creating and modifying objects.
 
 

https://www.jetbrains.com/datagrip/features/
License

DbSchema (by WiseCoders)

Visual Designer for Relational & NoSQL Databases You don't have to be an database expert to manage complex databases. DbSchema has everything you need to visually design, document and query relational and NoSQL databases.
 

https://dbschema.com/?AVGAFFILIATE=82405&__c=1
License

DataStax Studio (by DataStax)

Designed to facilitate Cassandra Query Language (CQL), Graph/Gremlin, and Spark SQL language development, DataStax Studio has all the tools needed for ad hoc queries, visualizing and exploring data sets, profiling performance and comes with a notebook interface that fuels collaboration..

https://www.datastax.com/dev/datastax-studio
OSS

Apache Zeppelin

Apache Zeppelin interpreter concept allows any language/data-processing-backend to be plugged into Zeppelin. Currently Apache Zeppelin supports many interpreters such as Apache Spark, Python, JDBC, Markdown and Shell..

https://zeppelin.apache.orgs
OSS

Jupyter Notebooks

JupyterLab is a interactive development environment for Jupyter notebooks, code, and data. JupyterLab is flexible: configure and arrange the user interface to support a wide range of workflows in data science, scientific computing, and machine learning.

https://jupyter.org/
Free trial

Razor SQL

The RazorSQL Apache Cassandra database client and query tool includes a Cassandra database browser, SQL editor, table editor, Cassandra import and export tools, Cassandra backup tools, and other custom Cassandra GUI tools

https://razorsql.com/features/cassandra_database...

3. Development libraries (ORM, Generators...)

OSS

Achilles

Achilles is an open source advanced object mapper for Apache Cassandra.
 

http://doanduyhai.github.io/Achilles/
Java
Deprecated

Astyanax

Astyanax is a high level Java client for Apache Cassandra.
 

https://github.com/Netflix/astyanax
Java
OSS

Casser

Fast and easy, functional style cutting edge Java 8 and Scala 2.11 Cassandra client.
 

https://github.com/noorq/casser
Java
OSS

Scalar DB

A library that makes non-ACID distributed databases/storages ACID-compliant. It not only supports strongly-consistent ACID transactions, but also scales linearly and achieves high availability when it is deployed with distributed databases/storages such as Cassandra.
 

https://github.com/scalar-labs/scalardb
Java
OSS

Kundera

Kundera is a "Polyglot Object Mapper" with a JPA interface. The idea behind Kundera is to make working with NoSQL Databases drop-dead simple and fun
 

https://github.com/Impetus/Kundera/wiki
Java
OSS

Play ORM

ORM for nosql with SCALABLE SQL
 

https://github.com/deanhiller/playorm
Java
OSS

Express Cassandra

Express-Cassandra is a Cassandra ORM/ODM/OGM for NodeJS with Elassandra & JanusGraph Support.

https://github.com/masumsoft/express-cassandra
Node JS (Javascript)
OSS

Spring Data Cassandra

The Apache Cassandra NoSQL Database offers many new capabilities for teams seeking a solution to handle high velocity, high volume and variable data flows..

https://spring.io/projects/spring-data-cassandra
Java
OSS

Ambien

Command line to generate a working JAVA project (Spring Boot) exposing a REST API for your Cassandra Classes.
 

https://github.com/brianmhess/ambien
Java
Deprecated

Caffinitas

Java object mapper for Apache Cassandra.
 
 
 

https://bitbucket.org/snazy/caffinitas/src/develop/
Java
OSS

CQLish

Lightweight CQL shell with an embedded Cassandra. Does not connect to an external Cassandra instance, (like cqlsh), but rather spins up an embedded Cassandra instance.
 

https://github.com/brianmhess/cqlish
Java
OSS

Cassandra Count

Simple program to count the number of records in a Cassandra table. By splitting the token range using the numSplits parameter, you can reduce the amount each query is counting and reduce the probability of timeouts.

https://github.com/brianmhess/cassandra-count
Java
OSS

Cassandra Loader

cassandra-loader is a general-purpose, delimited-file, bulk loader for Cassandra. It supports a number of configuration options to enable bulk loading of various types of delimited files.

https://github.com/brianmhess/cassandra-loader
Java
